# 淘淘网上商城
## 技术清单
* Spring,Springmvc,Mybatis
*	JSP、JSTL、JQUERY、EASYUI、KINDEDITOR（富文本编辑器）<br>
*	REDIS（缓存服务器，单点登录，购物车）<br>
*	SOLR（搜索）<br>
*	DUBBO（分布式服务框架）<br>
*	HTTPCLIENT（HTTP 协议访问客户端）<br>
*	ACTIVEMQ（消息队列）<br>
*	FASTDFS（图片服务器）<br>
*	NGINX（反向代理服务器）<br>
* Maven,Tomcat8,Jdk1.8,Eclipes
## 系统功能介绍
* 后台管理系统：管理商品、订单、类目、商品规格属性、用户管理以及内容发布等功能。<br>
* 前台系统：用户可以在前台系统中进行注册、登录、浏览商品、首页、下单等操作。<br>
* 购物车系统：用户可以在该系统中添加商品到购物车、修改购物车信息。<br>
* 订单系统：提供下单、查询订单。<br>
* 搜索系统：提供商品的搜索功能。<br>
* 单点登录系统：为多个系统之间提供用户登录凭证以及查询登录用户的信息。<br>
## 运行环境搭建
* 搭建Linux虚拟机百度云地址
* 下载nginx并修改conf下的nginx.conf
* 下载SwtichHost其中配置如taotaoHost.txt
